Chocoholics Anonymous System Analysis

Chocoholics Anonymous (ChocAN) is implementing a computerized system with a main module that includes separate user interfaces for operators, providers, and managers. Auxiliary modules will provide necessary functionality with access to ChocAn data center databases. User authentication will be stored in a user database, while EFT data will be stored in a financial database. A card reader and embedded computer will interact with the software. The system will interact with Acme Accounting Services and ChocAn Data Center for validation/authentication. Operators will access the user database, providers will interact with the Provider Directory, and managers will have access to the reports terminal. Project will take 3 months and use Java.
